Detailed Description

An unsaved new revision.

Most of its API is inherited from CBLRevisionBase.

Method Documentation

- (CBLRevision*) save: (NSError **)  outError

Saves the new revision to the database.

This will fail with a 412 error if its parent (the revision it was created from) is not the current revision of the document. Afterwards you should use the returned CBLRevision instead of this object.

Returns
A new CBLRevision representing the saved form of the revision.
- (void) addAttachment: (CBLAttachment *)  attachment

Creates or updates an attachment.

The attachment data will be written to the database when the revision is saved.

Parameters
attachmentA newly-created CBLAttachment (not yet associated with any revision)
